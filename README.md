## 关于我

凹语言的联合发明人，PLOC编程语言开放社区发起人，Go语言代码贡献者，曾作为蚂蚁高级技术专家获上海智慧工匠提名。是《国产编程语言蓝皮书》策划者、《Go语言圣经》翻译者、并出版《Go语言高级编程》《WebAssembly标准入门》等多部技术著作，是计算机和信息学技术的传播者。

## 将要去哪里

作为80后码农开始规划后半生的职业方向，希望未来能够继续参与到国产编程语言的探索和完善工作，同时未来国内计算机信息学编程的启蒙作一定的贡献。如果你也在做类似的事情欢迎交流。
