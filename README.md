## 关于我

柴树杉，80后码农，目前定居杭州。凹语言联合发起人，PLOC编程语言开放社区发起人，Go语言代码贡献者，曾获上海智慧工匠提名。是《国产编程语言蓝皮书》策划者、《Go语言圣经》翻译者、并出版《Go语言高级编程》《WebAssembly标准入门》等多部技术著作，2019年人民邮电出版社最具影响力作者，是开源技术实践者。

## 十年小目标(2018-2030)

- 小目标：写10本书+做一个非玩具的语言，简称非常10+1。
- 小进展：已正式出版5本书；凹语言走过5年已完成全部语言特性，处于生产试用阶段。
